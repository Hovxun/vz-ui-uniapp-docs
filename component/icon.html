Icon 图标 ​

基础用法 ​

html
<vz-icon name="vz-icon-home"></vz-icon>

图标颜色 ​

html
<vz-icon name="vz-icon-home" color="teal"></vz-icon>

图标大小 ​

html
<vz-icon name="vz-icon-home" size="50rpx"></vz-icon>

图片图标 ​

html
<vz-icon name="/static/logo.png"></vz-icon>

自定义图标 ​

html
<vz-icon name="iconfont icon-setup"></vz-icon>

iconfont-阿里巴巴矢量图标库

css
@font-face {
  font-family: "iconfont";
  src: url("...");
}

.iconfont {
  font-family: "iconfont";
}

.icon-setup::before {
  content: "\e617";
}

属性名说明类型默认值样式变量

color图标颜色stringinherit--vz-icon-color
size图标大小string / numberinherit--vz-icon-size
i-class根节点样式类string--
